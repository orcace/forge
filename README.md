# Forge

A fast, privacy-first developer toolbox built with React + TypeScript.

---

## Features

* Common dev tools (JSON, Diff, JWT, Base64, ...)
* Fast & lightweight (lazy-loaded)
* Keyboard-first UX
* Runs entirely in the browser

---

## 🚀 Getting Started

```bash
git clone https://github.com/cuthanhcam/forge.git
cd forge
pnpm install
pnpm dev
```

---

## 🧱 Tech Stack

* React + TypeScript
* Vite
* Zustand
